##  System Overview

This system simulates core banking operations including:

- Customer onboarding
- Account management (Savings / Current / Salary)
- Secure fund transfers
- Credit & Debit card management
- Loan processing & repayments
- Branch and staff management

The system is designed with proper relational database constraints and transaction safety.
